# Base Conversion
CSC16 Project 1

Overview
	Your goal is to create a program that will convert from any base to base 10. All digits should work, the alphabet will be added in the future.

Summary
	This program can convert from any base to base 10. During the time of posting, this program will only accept digits. The alphabet will be something we will add in the future
